# RVY Solutions Website Design

This folder contains a static website layout inspired by the RVY Solutions ER diagram.

## What is included

- `index.html` – Frontend landing page with modules for Home, About, Services, Brands, Products, Gallery, Testimonials, and Contact.
- `admin.html` – Admin dashboard skeleton for user, brand, product, service, order, support, and analytics management.
- `login.html` – Login page with a clean sign-in form.
- `signup.html` – Signup page for new account creation.
- `styles.css` – Responsive styling aligned to a modern business website.
- `script.js` – Navigation toggle and contact form/auth form interaction.

## Design Mapping from ER Diagram

- Users → contact form, testimonials, and potential admin user management screens.
- Brands → brand showcase section and admin brand management.
- Services → service cards and service management panel.
- Products → product catalog cards and product management dashboard.
- Categories → product catalog grouping and admin category concept.
- Orders → order management table in admin dashboard.
- Contact Messages → contact form and customer support section.
- Gallery → gallery section and gallery upload admin module.
- Testimonials → review section.
- Newsletter Subscribers → subscriber metrics in admin panel.

## How to preview

1. Open `index.html` in your browser to view the public site.
2. Open `admin.html` to view the admin dashboard layout.
